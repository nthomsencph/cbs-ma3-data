## Format

All files are markdown with `#`/`##` heading structure. The largest document is `exam-project.md` (1,162 lines) covering written assignment and project exam rules. The smallest are the contact pages (~27-39 lines).

## Source

Documents were collected from public CBS student-facing pages at [cbs.dk](https://www.cbs.dk/) and the [CBS course catalog](https://kursuskatalog.cbs.dk/). Content reflects the 2025/2026 academic year.
